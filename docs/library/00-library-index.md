# GazaarY Library – Canon Index

This folder is the **single source of truth** for important decisions in the GazaarY site.

It is meant to survive chat deletions, refactors und Experimente.

## Aktuelle Canon-Dokumente

- `hero-steps-canon.md`  
  Canonical Hero Section:
  - `src/components/HeroSteps.tsx`
  - 3-Step Hero mit “Kiss” in die KOA Section
  - Immer helle Navigation, keine alten “muting” Varianten

- `koa-section-canon.md`  
  Canonical KOA Section:
  - `src/components/KOASection.tsx`
  - Smart Surface (Lazy Susan) mit GazaarY Y-Core
  - Gespiegelte Booklets, responsive Tischgröße, aktueller deutscher Text

- `working-style-truth-protocol.md`  
  Wie hier gearbeitet wird:
  - Truth Protocol (Genauigkeit, Quellen, keine Fantasiezahlen)
  - Coding Style (Full-File-Rewrites, UX-Fokus)
  - Tech-Stack & Deploy (Next.js, Tailwind, Vercel)

## Nutzung der Library

- Bevor du Hero oder KOA anfasst:
  - Das passende Canon-Dokument lesen.
- Nach **größeren Änderungen**:
  - Das Canon-Dokument in diesem Ordner anpassen.
- Bei Zweifel:
  - Diese Library gewinnt gegen alte Screenshots, alte Chats oder Erinnerungen.
